VPS-Centos 7搭建配置Shadowsocks科学上网

刚刚入手了一个VPS，拿来练手Linux类系统，最重要的还是顺便自己搭建SS科学上网。我这里的是Centos 7 64位。

安装

# yum install python-setuptools && easy_install pip
# pip install shadowsocks
这样SS就安装好啦，然后我们要配置一下。

配置

新建一个json文件
# vim /etc/shadowsocks.json
然后写入配置，端口和密码自己自定义。

{
    "server":"0.0.0.0",
    "server_port":8900,
    "local_address": "127.0.0.1",
    "local_port":1080,
    "password":"passwd",
    "timeout":300,
    "method":"aes-256-cfb",
}

如果要多用户登陆，那么就把password改成port_password然后加入多个端口和密码，例如： 

{
    "server":"0.0.0.0",
    "local_address":"127.0.0.1",
    "local_port":1080,
    "port_password":{
         "8900":"passwd0",
         "8901":"passwd1",
         "8902":"passwd2",
         "8903":"passwd3",
         "8904":"passwd4"
    },
    "timeout":300,
    "method":"aes-256-cfb",
>

然后新建一个ss服务
# vim /etc/systemd/system/shadowsocks.service
内容写入： 

[Unit]
Description=Shadowsocks

[Service]
TimeoutStartSec=0
ExecStart=/usr/bin/ssserver -c /etc/shadowsocks.json

[Install]
WantedBy=multi-user.target

然后开启服务，并让服务开机启动
# systemctl enable shadowsocks
# systemctl start shadowsocks
这时候可以检查一下服务是否正确开启
# systemctl status shadowsocks
如果Active那里显示的是绿色的active(running)则说明服务正确开启了。如图：

最后，千万记得防火墙开启刚才配置里写的端口，我的例子里写的只有一个8900，防火墙没开的请自动忽略。
# firewall-cmd --add-port=8900/tcp
查看端口是否正确开启
# firewall-cmd --query-port=8900/tcp
返回success则正确开启了，Enjoy it！
